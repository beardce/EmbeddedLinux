To run the modified matrixLED.js file set up socket.io by making the calls
cd $NODE_PATH
sudo ln -s bonescript/node_modules/socket.io

if you have already set up socket.io then just run boneServer

sudo node boneServer.js

after that browse to 192.168.7.2:9090 and click on the matrixLED panel.
